Exchange

不出国原因：

  1. 源源，爸爸，奶奶。难回来
  2. 留不下来，身份问题难解决，且要花倾尽所有的钱。不想为身份而提心吊胆，不想为花了很多父母的钱而有心理压力
  3. 学校绩点制度不友好，且我不认可国外看重GPA的模式
  4. 美国20W刀，算上购买力按1:3计算，相当于国内60W，还要算上房租和税，这意味着我只要在国内找到3、40W的工作就满足了
  5. 部分爱国情怀

唯一担心的只是年龄问题

不交换UCB原因：

  1. 源源

  2. 交换只是赚钱的生意，性价比不如留学。（在只能付出交换的钱而不是有远远大于交换的钱的情况下）

  3. 感觉自己不够社牛，不可能到了国外就指望一切都能突然转变过来。我内心不耻这种在一千人的大课中去讨好陶瓷教授以获取推荐信的行为，过于目的性

  4. 天意，六级没过520，我也是刚发现

以上两条都加一条相同的原因：没钱——不想花别人的钱

想去国外名校其实就和想买苹果手机一样——贪婪

——2024.3.21

conclusion

  1. 不用焦虑27岁，练好技术是王道，毕竟有985CS的招牌

  2. 担心35岁职业危机？多探索副业是关键

  3. 国外没有那么好，国内没有那么差，国内也没有那么好

——2024.3.21

2024.4.2

如何安排时间

  1. 周一到周五：学课外
  2. 周末：和源源玩，留半天学习课内（速刷）和写作业
  3. 寒暑假：源源不在时疯狂找补，源源在时和源源玩
  4. 碎片时间：见缝插针地弥补

  • 代价：不重要的GPA和老师的压力

  • 可以获得：宝贵的时间轮转，借用未来的时间

  • 关于和源源玩是宅着还是出去玩：看到有好玩的、合适的就出去玩

  • 关于王者荣耀等游戏：除王者荣耀，别的游戏不玩；王者荣耀和源源玩时需抛弃好胜心，只是陪伴和娱乐。

——2024.4.2

大二下课表分析

  • 注重实际！

  • ==只要保持持续的努力，其实时间怎么分配，花在哪里并不重要==

  • ==有时候一点点时间不重要，有时候一点点时间都很重要==

  • 去上课就能保证不违反规则，因为没有任何一条制度规定了上课不好好听课就会怎样怎样

  • 去的好处：

    • 能大概知道老师讲到了哪
    • 不违反制度
    • 如果点名能参与
    • 不会被源源骂TAT
    • 可以换个环境，有助于放松，避免一直在图书馆
    • 会和源源暂时的分离，两个人也不能一直黏在一起。短暂的分离更有助于感情。

  • 不去的好处：

    • 节约一些时间
    • 避免老师巡视、提问的紧张和不便
